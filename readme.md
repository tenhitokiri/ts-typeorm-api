# TypeORM Nodejs REST API CRUD

## About

This is a simple Nodejs REST API CRUD example using:

- NodeJS
- Typescript
- Express
- TypeORM
- MySQL

## Information

For more information about TypeORM, please visit the [TypeORM website](https://typeorm.io/).
